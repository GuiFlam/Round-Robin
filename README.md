# Round-Robin Simulator

Simulateur d'ordonnancement Round-Robin en C, dans le cadre du cours ELE216.

## Structure du projet

```
.
├── main.c               # Point d'entrée du programme
├── includes/
│   └── processus.h      # Interface du module processus
├── src/
│   └── processus.c      # Implémentation du module processus
└── Makefile
```

## Compilation

```bash
make
```

## Exécution

```bash
./round_robin
```

## Nettoyage

```bash
make clean
```
